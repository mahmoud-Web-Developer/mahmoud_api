## 🌐 الـ Base URL
https://mahmoudapi-production.up.railway.app

## 📋 تصنيف الـ APIs

### 🔐 1. Authentication APIs (APIs المصادقة)

#### POST /auth/signup
- **الوصف**: تسجيل مستخدم جديد
- **البيانات المطلوبة**:
  ```json
  {
    "username": "string",
    "email": "string",
    "password": "string",
    "role": "user|admin"
  }
  ```
- **الاستجابة**:
  ```json
  {
    "success": true,
    "message": "User registered successfully",
    "data": {
      "user": {
        "id": "number",
        "username": "string",
        "email": "string",
        "role": "string"
      }
    }
  }
  ```

#### POST /auth/login
- **الوصف**: تسجيل الدخول والحصول على JWT
- **البيانات المطلوبة**:
  ```json
  {
    "username": "string",
    "email": "string",
    "password": "string"
  }
  ```
- **الاستجابة**:
  ```json
  {
    "success": true,
    "message": "Login successful",
    "data": {
      "user": {
        "id": "number",
        "username": "string",
        "email": "string",
        "role": "string"
      },
      "token": "JWT_TOKEN"
    }
  }
  ```

## 🔑 Authentication

### JWT Token
- **النوع**: Bearer Token
- **المدة**: 24 ساعة
- **الاستخدام**: في Header
  ```
  Authorization: Bearer JWT_TOKEN
  ```

### الأدوار المتاحة:
- **user**: مستخدم عادي
- **admin**: مدير النظام

## 📝 أمثلة الاستخدام

### تسجيل مستخدم جديد:
```bash
curl -X POST https://mahmoudapi-production.up.railway.app/auth/signup \
  -H "Content-Type: application/json" \
  -d '{
    "username": "testuser",
    "email": "test@example.com",
    "password": "password123",
    "role": "user"
  }'
```

### تسجيل الدخول:
```bash
curl -X POST https://mahmoudapi-production.up.railway.app/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "testuser",
    "password": "password123"
  }'
```

### جلب الخدمات:
```bash
curl -X GET https://mahmoudapi-production.up.railway.app/services
```
